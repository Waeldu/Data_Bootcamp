# Data_Bootcamp
WorkBooK_1 Summary

•	Legal Compliance: Regulations governing data security and privacy, such as GDPR and the Data Protection Act, are examined.

•	Data Manipulation: Practical Excel skills for managing datasets, including table formatting and the use of functions, are developed.

•	Advanced Analysis: Techniques for summarising complex data through Pivot Tables and the SWITCH function are acquired.

•	Professional Presentation: Strategies and tools for the preparation and delivery of data-driven insights to executive stakeholders are mastered

Workbook_2 Summary

•	Tableau Ecosystem: The distinct purposes and environments of various Tableau versions, such as Desktop, Server, and Public, are compared and contrasted.

•	Interactive Visualisation: Dashboards featuring geographical maps and bar charts are created using specialised datasets like UK job change statistics.

•	Analytical Trends: Deep-dive analyses on diverse datasets, including Spotify trends and NHS health data, are conducted to support organisational decision-making.

•	Power BI Reporting: Practical skills in report design and dashboard creation are developed through hands-on labs within the Power BI Desktop application.

Workbook_3 Summary

•	Database Structure: Fundamental concepts such as primary and foreign keys, along with various relational models (1:1, 1:M, and M:M), are explored.

•	Database Comparison: The core differences between relational (SQL) and non-relational (NoSQL) systems regarding schema flexibility and data integrity are analyzed.

•	SQL Querying: Advanced data retrieval methods, including multiple types of JOIN operations and filtering techniques, are mastered. 

•	System Management: The complete lifecycle of a database, from schema design and implementation using SQL commands to maintenance and security protocols, is detailed.

![image alt]{ https://github.com/Waeldu/Demo171125/blob/main/workbook_Image_3.jpg}

Workbook_4 Summary

•	Cloud Architecture: Essential cloud service models (IaaS, PaaS, SaaS) and various deployment strategies, such as Public and Hybrid clouds, are analyzed.

•	Legal Compliance: Critical UK legislations, including the Computer Misuse Act and Data Protection Act 2018, are examined in the context of organizational data security.

•	Azure Data Management: Practical skills for managing relational and non-relational data systems are developed through hands-on labs within the Microsoft Azure platform.

•	Data Strategy Application: Methods for modernizing manual data collection into scalable, cloud-based infrastructures are evaluated through a comprehensive business case study.

![image alt]{ https://github.com/Waeldu/Demo171125/blob/main/workbook_Image_4.jpg}

Workbook_5 Summary

•	Python Programming: Core programming logic and syntax are practised through the implementation of algorithmic challenges like "FizzBuzz".

•	Pandas Data Handling: Proficiency in reading, exploring, and indexing CSV data using the Pandas library is established.

•	Data Transformation: Advanced techniques for manipulating DataFrames, including column renaming, filtering, and conditional logic, are applied.

•	Visualisation and Export: Skills for visualising analytical results and exporting processed data for professional portfolio development are acquired.
